## Files changed

- `src/commands/grab.ts`: added `estTokensIn: number` to `GrabPlan`; imports `SYSTEM_PROMPT`; accumulates `contentChars` in scan loop; computes estimate; renamed inner `totalChunks` to `chunksToProcess` to avoid shadowing.
- `src/extractor/host-agent.ts`: exported `agentBrand(id)` and `detectAgentCli`; added `model?: string` to `HostProviderOpts`; `buildClaudeArgs` now accepts and prepends `--model <model>` when set.
- `src/extractor/index.ts`: passes `model: opts.config?.model` to `createHostProvider`.
- `src/extractor/local-model.ts`: imports `detectAgentCli` + `agentBrand`; preflight block message names the detected agent if one is on PATH.
- `bin/handprint.ts`: imports `agentBrand`; adds `humanTokens()` and `extractorLabel()` and `estimateLine()` helpers; `printScope` now shows branded extractor + estimate; `printScope` moved before the blocked-reason check so blocked runs still show the plan.
- `tests/commands/grab.test.ts`: added test asserting `plan.estTokensIn > 0`.
- `tests/extractor/host-agent.test.ts`: added `agentBrand` unit tests (claude, opencode, codex, unknown) and a `--model` threading test.
- `README.md`, `AGENTS.md`, `CHANGELOG.md`: documented `extraction.model` for host engine; noted token estimate and branded extractor label in plan.
